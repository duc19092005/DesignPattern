# Singleton Design Pattern ví dụ ở bên Typescript

## Về khái niệm : Singleton Design Pattern đảm bảo được tính độc nhất của 1 instance (Object) giả sử bạn có 1 lớp database bạn muốn dùng nó 1 lần trong chu kì app chạy ví dụ bạn cần chạy backend service liên tục và chỉ dùng 1 database thì khi này bạn sẽ áp dụng singleton để đảm bảo database chỉ là duy nhất
